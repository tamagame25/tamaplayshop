TamaPlayShop – Pacchetto pronto (FINAL)
======================================

Come vedere l’anteprima:
1) Apri `index.html` in un browser (PC o telefono).
2) Sostituisci i file in `assets/` con i tuoi (stessi nomi) per vedere subito i loghi/cover reali.

Sostituzioni:
- assets/tama-logo.png           -> Logo TamaPlayShop (PNG/SVG)
- assets/rewardrush-logo.png     -> Icona/logo Reward Rush
- assets/egiziacity-cover.jpg    -> Cover EgiziaCity
- assets/pollaio-cover.jpg       -> Cover Pollaio Challenge
- assets/boardhub-cover.jpg      -> Cover sezione Giochi da Tavolo

Tutto è già responsive, con:
- Banner pubblicitario
- Hero Reward Rush con CTA
- Voci Giochi / Premi / Stanza Relax
- Macro pulsanti: Gioca e guadagna + Giochi da tavolo (tornei & vincite)
- Griglia Giochi (pulsante Gioca/Regole)
- Teaser Stanza Relax
